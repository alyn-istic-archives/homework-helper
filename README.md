# 🌊 wave: new motion 🌊
**wave: new motion** is a studying website that tracks your study time in real time through a stopwatch for periods of time spent studying, and rewards effort in the form of a personal shop!


## ♒︎ Features ♒︎

- Able to create new sessions through using buttons with reference screenshots
- **Time**
  - Built in Timer and Stopwatch
  - Logs time to create new sessions

---

## 𖦹 The Timer - How to Use 𖦹

1. Set the stopwatch for an unlimited time session
2. After a couple hours using the stopwatch, press save and a popup appears
3. Save course name/title, hours spent studying, and details on the session
4. Use hours spent to buy personal items, fulfill personal goals, etc.,

---


## The Shop - How It Works
The shop is a personal reward system!!
1. You set your own items or goals. (e.g. You want to reward yourself with a scroll break after an hour of studying)
(The shop doesn't come pre-filled with any items. Everything in it is created by you using the New Goal/Item button. Like a wishlist, based of personal motivation!)
2. When setting items, they will have a cost in hours, and be either one-time or renewable. 
  When you create a new goal or item, you assign it an hour cost. Once you've studied long enough to earn that many hours, you'll be able to purchase it.
  One time items disappear from the shop and show up under 'show bought' if you'd like a history of purchased items!
  renewable items stay in the shop, but you can also see how many times you've purchased them in show bought

## Show Shop vs. Show Bought
Show Shop displays all your unbought items — things you've added but haven't purchased yet. If you just created a goal and haven't bought it, it will appear here.
Show Bought displays everything you've already purchased.
If both views look the same or empty, it just means you haven't created any goals yet, or you've already bought everything!


## Not enough hours?
If you try to buy something you can't afford yet, the app will let you know. Keep studying to earn more hours!

## Clear Bought
The Clear Bought button wipes your purchase history so you can start fresh.
